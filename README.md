# ENRI-Resin-Duct-Tools
A suite of three FIJI/ImageJ macros to aid in quantifying tree defense characteristics using scans of tree cores. Developed at the Environment and Natural Resources Institute (ENRI), University of Alaska Anchorage and funded through a Joint Venture Agreement with the USDA Forest Service (25-JV-11261918-076). See ENRI_resin_duct_protocol.pdf for detailed instructions on how to use the macros.
